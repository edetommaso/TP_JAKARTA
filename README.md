# Difficultés rencontrées lors du TP Jakarta EE

Ce TP a présenté quelques défis, principalement liés à :

1.  **L'utilisation de `PreparedStatement` :** La manipulation des `PreparedStatement` en Java/JDBC a été un point de friction, notamment en raison de sa ressemblance avec la manière dont les requêtes préparées sont gérées en PHP. Cela a parfois conduit à une confusion dans l'approche à adopter pour la gestion des paramètres.

2.  **L'initialisation de la base de données :** La mise en place et l'initialisation de la base de données ont également nécessité une attention particulière pour assurer le bon fonctionnement de l'application.
