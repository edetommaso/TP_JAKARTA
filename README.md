# TP Jakarta - Organisation du projet

Ce dépôt contient l'ensemble des travaux pratiques (TP) réalisés. Chaque TP est organisé dans une branche spécifique pour faciliter le suivi et la séparation des technologies.

## Structure des branches

### Java EE
Les travaux pratiques concernant Java EE se trouvent dans les branches suivantes :
- `TP/1`
- `TP/2`
- `TP/3`
- `TP/4`

### Intelligence Artificielle (IA)
Le TP portant sur l'IA est disponible dans la branche :
- `TP/IA/1`

### Blockchain
Le TP concernant la Blockchain est disponible dans la branche :
- `TP/BlockChain/1`

---

Pour accéder à un TP spécifique, veuillez basculer sur la branche correspondante :
```bash
git checkout <nom-de-la-branche>
```
